# Getting Started

## Installation

Vue Stripe is easy to install. It is available in node package manager so you install it direcly via **Yarn** or **NPM**.

:::: code-group
::: code-group-item Yarn
```sh
yarn install @vue-stripe/vue-stripe
```
:::
::: code-group-item NPM
```sh
npm install @vue-stripe/vue-stripe
```
:::
::::
